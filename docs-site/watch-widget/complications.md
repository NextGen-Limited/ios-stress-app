# Complications

StressMonitor provides watch complications in multiple sizes to fit different watch faces.

## Available Complication Sizes

| Size | Display |
|------|---------|
| Graphic Corner | Stress color arc + level label |
| Graphic Circular | Stress color ring with icon |
| Graphic Bezel | Full-width stress gauge |
| Modular Small | Color indicator + number |
| Modular Large | Stress label + last updated time |
| Utilitarian Small | Minimal stress icon |

## What Each Complication Shows

All complications display:
- **Current stress level** (color-coded: green / blue / yellow / red)
- **Stress category label** (Excellent / Normal / Attention / Overload)

Larger complications also show:
- Last measurement time
- Numeric stress score (0–100)

## Color Coding

| Color | Category |
|-------|----------|
| 🟢 Green | Excellent State |
| 🔵 Blue | Normal State |
| 🟡 Yellow | Attention Needed |
| 🔴 Red | Pressure Overload |
