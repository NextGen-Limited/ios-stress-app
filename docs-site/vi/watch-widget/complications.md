# Complications

StressMonitor cung cấp complications ở nhiều kích thước để phù hợp với các mặt đồng hồ khác nhau.

## Kích thước complication có sẵn

| Kích thước | Hiển thị |
|-----------|---------|
| Graphic Corner | Cung màu căng thẳng + nhãn cấp độ |
| Graphic Circular | Vòng màu căng thẳng với biểu tượng |
| Graphic Bezel | Thanh đo căng thẳng toàn chiều rộng |
| Modular Small | Chỉ báo màu + số |
| Modular Large | Nhãn căng thẳng + thời gian cập nhật cuối |
| Utilitarian Small | Biểu tượng căng thẳng tối giản |

## Mỗi complication hiển thị gì

Tất cả complications hiển thị:
- **Cấp độ căng thẳng hiện tại** (màu sắc: xanh lá / xanh dương / vàng / đỏ)
- **Nhãn danh mục** (Xuất sắc / Bình thường / Chú ý / Quá tải)

Complications lớn hơn còn hiển thị:
- Thời gian đo gần nhất
- Điểm căng thẳng dạng số (0–100)

## Mã màu

| Màu | Danh mục |
|-----|---------|
| 🟢 Xanh lá | Trạng thái xuất sắc |
| 🔵 Xanh dương | Trạng thái bình thường |
| 🟡 Vàng | Cần chú ý |
| 🔴 Đỏ | Quá tải áp lực |
